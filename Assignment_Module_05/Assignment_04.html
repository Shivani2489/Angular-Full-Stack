2. Steps to Create HTML File in VS Code :

  1. Create a Folder => Name it HTML
  2. Type cmd in folder's path to go to Command Prompt
  3. Type code . in cmd and Enter
  4. Press CTRL(Control) key + N to create new file
  5. Give relevant name to that file with extention .html
